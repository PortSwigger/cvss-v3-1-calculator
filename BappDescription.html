CVSS v3.1 Calculator is a Burp Suite extension that enables fast and accurate calculation of CVSS (Common Vulnerability Scoring System) v3.1 base scores. It streamlines the vulnerability assessment process by offering a user-friendly interface for selecting metric values, making it easier for security professionals to determine and communicate the severity of identified issues.

Features

  • Base Metrics Calculation: Accurately compute CVSS v3.1 base scores by selecting metric values through an intuitive interface.
  • Interactive UI: Fully embedded within Burp Suite, allowing for seamless integration into your existing assessment workflow.
  • Tabbed Interface: Includes a dedicated tab labeled "CVSS v3.1" for easy navigation and focused score calculation.

Usage

  1. Navigate to the "CVSS v3.1" tab within Burp Suite.
  2. Select the appropriate values for each CVSS base metric.
  3. The extension will automatically calculate and display the resulting base score based on your selections.
